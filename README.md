# 🏠 Nexus Inmobiliaria - Panel Seguro v1.0

Sistema de gestión inmobiliaria diseñado para automatizar el registro de clientes y garantizar la seguridad en el contacto mediante códigos de validación únicos.

## 🚀 Características
- **Registro Seguro**: Generación de códigos REF únicos para evitar estafas.
- **Multicanal**: Botones de contacto directo vía WhatsApp y Telegram.
- **Roles de Acceso**: Secciones protegidas para Vendedores y Administradores.
- **Portabilidad**: Base de datos ligera en SQLite y reportes en Excel.

## 🛠️ Tecnologías Usadas
- **Lenguaje**: Python 3.x
- **Framework**: Streamlit
- **Base de Datos**: SQLite3
- **Análisis**: Pandas / Openpyxl

## 📦 Instalación (Local)
1. Clonar el repositorio.
2. Crear entorno virtual: `python -m venv .venv`
3. Activar: `source .venv/bin/activate`
4. Instalar dependencias: `pip install -r requirements.txt`
5. Correr: `streamlit run main.py`
